### wechatPay-API-v3官方文档：
https://wechatpay-api.gitbook.io/wechatpay-api-v3/
全新的微信支付API v3，相较于的之前微信支付API，主要区别是：
- 遵循统一的Restful的设计风格；
- 使用JSON作为数据交互的格式，不再使用XML；
- 使用基于非对称密钥的SHA256-RSA的数字签名算法，不再使用MD5或HMAC-SHA256；
- 不再要求HTTPS客户端证书；
- 使用AES-256-GCM，对回调中的关键信息进行加密保护
