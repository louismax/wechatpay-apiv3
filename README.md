## 安装
`go get -v github.com/louismax/wxpayv3`

## 实现能力
### 商户进件(仅支持服务商)
- 提交申请单-  ❌
- 查询申请单状态  ✔️
- 修改结算账号  ❌
- 查询结算账号  ✔️
